# Geass Anime Website
MERN STACK (HTML/CSS,React.js, Node.js, MongoDB) | 
Features:
- Add Anime to WatchList
- Delete Anime 
- MongoDB
- React
- Node.js / Express.js

# Run Web App
- npm install to install all the node modules
- direct yourself to src/backend , to run server : nodemon server.js
- to run front end npm start
